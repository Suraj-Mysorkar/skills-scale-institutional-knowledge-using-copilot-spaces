# Personas / Roles - Review Checklist

Use this checklist when adding new personas or updating existing ones to ensure consistency and usefulness.

- [ ] Role Summary present and concise
- [ ] Clear list of Responsibilities
- [ ] Interactions with existing roles documented
- [ ] Typical artifacts or deliverables listed
- [ ] Links to related process docs or runbooks
- [ ] Reviewed by at least one existing role owner (PM, Dev, or QA)
- [ ] Acceptance criteria for key handoffs included where applicable
